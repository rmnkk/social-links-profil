## Project setup

If you want to test the project on your own machine

### Instal dependencies
```
pnpm install
```

### Compile and Hot-Reload for Development
```
pnpm dev
```

### Type-Check, Compile and Minify for Production
```
pnpm build
```

### Lint with [ESLint](https://eslint.org/)
```
pnpm lint
```
